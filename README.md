## ğŸ“ Customization

### Mengubah Nomor WhatsApp
Edit baris ini di `index.html`:
```html
<a href="https://wa.me/6281234567890?text=..." 
```
Ganti `6281234567890` dengan nomor WhatsApp Anda.

### Mengubah Lokasi Google Maps
Edit bagian iframe Google Maps:
```html
<iframe 
    src="https://www.google.com/maps/embed?pb=..." 
```
Ganti dengan embed URL Google Maps lokasi showroom Anda.

### Mengubah Email Contact
Edit di bagian footer:
```html
<p class="text-[#A0A0A0] text-sm mb-2">Email: vip@obsidian-auto.com</p>
```

### Menambah/Mengubah Mobil di Inventory
Edit bagian "Featured Inventory Section" dan tambahkan card baru dengan struktur yang sama.
